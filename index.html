Descripción del prototipo

Sistema IoT portátil basado en NVIDIA Jetson Nano para adquisición georreferenciada de imágenes en zonas con conectividad limitada.

El dispositivo captura fotografías, obtiene coordenadas GPS y sincroniza automáticamente la información con la nube cuando detecta conexión.

Puede utilizarse en monitoreo ambiental, inspección de infraestructura, supervisión de vías y adquisición de datos en campo.

Instrucciones de uso

1. Encienda el dispositivo.

2. Espere señal GPS.

3. Presione el botón de captura.

4. La imagen y ubicación se almacenan automáticamente.

5. Si hay internet, el sistema sincroniza los datos automáticamente.

6. Use el botón de galería para visualizar las imágenes.
